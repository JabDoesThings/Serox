Serox V2.00 Release

This is a tool to take SubSpace LVL format maps and export a given area as a image to use for 
LVZ purposes.

Usage: [x1] [y1] [x2] [y2] [lvl] [save dir]
NOTE: coordinates can only be 0-1023.

To use mask features, create a .lvl file with the same name, followed by '_mask'.
